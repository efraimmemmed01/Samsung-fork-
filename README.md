# Samsung Saytının Klonu (React + Vite)

Bu layihə orijinal [Samsung US](https://www.samsung.com/us/) ana səhifəsinin mükəmməl (pixel-perfect) klonudur. Layihə müasir veb texnologiyalarından istifadə edilərək, interaktiv və sürətli bir istifadəçi təcrübəsi təqdim etmək məqsədilə hazırlanmışdır.

## 🚀 Texnologiyalar

*   **React** - Komponent əsaslı quruluş üçün.
*   **Vite** - Yüksək sürətli development mühiti üçün.
*   **Tailwind CSS** - Müasir, sürətli və responsiv dizaynlar üçün.
*   **Framer Motion** - Səlis skroll və vizual animasiyalar üçün.
*   **Lottie** - Veb səhifədə yüksək keyfiyyətli animasiyalar üçün.

## ✨ Əsas Xüsusiyyətlər

*   **Mükəmməl Dizayn:** Header, Hero, Məhsul Seçimləri və Footer daxil olmaqla bütün detallar orijinalına uyğun olaraq dizayn edilmişdir.
*   **Mikro-interaksiyalar:** Elementlər üzərinə gəldikdə (hover effect) və kliklərdə xüsusi reaksiyalar və zərif animasiyalar.
*   **Tam Təchizatlı Modallar:** Axtarış (Search), səbət (Mini Cart) və məhsula sürətli baxış ekranları.
*   **Skeleton və Parallax Yükləmələr:** Səhifə yüklənərkən Skeleton Loader-lər istifadə edilmiş və səhifə içi Parallax effektlər tətbiq olunmuşdur.
*   **Responsivlik:** Bütün cihaz ölçülərində mükəmməl görünüş.

## ⚙️ Quraşdırma və İşə Salma

Layihəni lokal mühitinizdə işə salmaq üçün:

1. **Repozitoriyanı klonlayın:**
   ```bash
   git clone https://github.com/efraimmemmed01/Samsung-fork-.git
   ```
2. **Qovluğa daxil olun:**
   ```bash
   cd Samsung-fork-/samsung
   ```
3. **Paketləri yükləyin:**
   ```bash
   npm install
   ```
4. **Layihəni başladın:**
   ```bash
   npm run dev
   ```

## 📝 Əlavə Məlumat

Bu layihədə bütün lazımsız (node_modules, lazımsız build faylları və s.) fayllar `.gitignore` ilə gizlədilmişdir ki, repozitoriya təmiz qalsın. Layihə portfel üçün hazırlanmışdır və lokal aktivlərlə (şəkillər, ikonlar) tam optimallaşdırılmışdır.
