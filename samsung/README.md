# Samsung Clone (React + Vite)

Bu layihə orijinal Samsung vebsaytının React və Vite istifadə olunaraq hazırlanmış klonudur. Layihə tam olaraq müstəqil komponentlərə ayrılmış, müasir görünüş və interaktivlik qazandırmaq məqsədilə Framer Motion və Tailwind CSS kimi vizual alətlər ilə dəstəklənmişdir.

## 🚀 Texnologiyalar

- **React:** UI komponentlərinin qurulması üçün
- **Vite:** Sürətli inkişaf mühiti (development environment) və build prosesi üçün
- **Tailwind CSS:** Üslublandırma (styling) və responsive dizayn üçün
- **Framer Motion:** Axıcı və müasir animasiyalar üçün
- **React Router DOM:** Səhifələrarası yönləndirmə (routing) üçün

## 🌟 Xüsusiyyətlər (Features)

- **Parallax Effect:** Səhifədə aşağı getdikcə arxa fon və obyektlərin fərqli sürətdə hərəkət etməsi.
- **Skeleton Loaders:** Şəkil və məzmunlar tam yüklənənə qədər müvəqqəti yükləmə ekranlarının (iskelet konstruksiya) göstərilməsi.
- **Micro-interactions:** Düymələrin üzərinə gəldikdə və toxunduqda incə animasiyaların işə düşməsi. (Hover & Tap effect)
- **Lottie Animasiyaları:** Səhifədə modern və yüngül animasiyaların yerləşdirilməsi.
- **Cart & Quick View Modals:** Məhsullara sürətli baxış pəncərəsi (modal) və yan tərəfdən açılan səbət hissəsi.
- **Search Modal:** Tam ekran və interaktiv axtarış paneli.
- **Mobile Responsive Navbar:** Mobil cihazlar üçün optimallaşdırılmış "hamburger" menyu.

## 🛠️ Quraşdırma və Çalışdırma (Local environment)

Layihəni öz kompüterinizdə çalışdırmaq üçün aşağıdakı addımları izləyin:

1. **Repozitoriyanı klonlayın:**
```bash
git clone https://github.com/Raemishex/samsung-fork.git
cd samsung-fork
```

2. **Paketləri yükləyin:**
```bash
npm install
```

3. **Layihəni başladın:**
```bash
npm run dev
```

4. **Brauzerdə açın:** Terminalda sizə təqdim olunan linkə (adətən `http://localhost:5173`) daxil olun.

## 🌐 Deploy (Vercel)

Layihə artıq Vercel uyğun olaraq qurulmuşdur (bunun üçün daxilində `vercel.json` faylı mövcuddur). Layihəni Vercel platformasında canlıya çıxarmaq sadədir.
- Vercel-ə GitHub hesabınızla daxil olun.
- "Add New Project" seçin.
- `samsung-fork` repozitoriyasını seçərək "Import" vurun.
- Framework Preset olaraq "Vite" seçiləcək. (Əgər Build qovluqlarının avtomatik seçildiyini görsəniz dəyişdirməyin).
- "Deploy" düyməsinə klikləyin.
